# Helpdesk Internal Custom

Custom Frappe app for internal departmental helpdesk with team-based permissions.

## Features

- Dual-role permission system (Agent/Customer view)
- Team-based ticket access control
- Custom fields for team management
- Pre-built onboarding data via fixtures

## Installation

```bash
bench get-app https://github.com/sthalatech/helpdesk_internal_custom
bench --site [site-name] install-app helpdesk_internal_custom
```
